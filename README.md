# M3U8 Proxy Demo (优化版)

一个功能完善的M3U8流媒体代理服务器，支持广告过滤、路径重写和性能优化。

## 主要改进

1. **配置外部化** - 所有设置都在 `config.js` 中管理
2. **模块化架构** - 核心功能拆分为独立模块
3. **增强错误处理** - 更详细的错误信息和状态码
4. **缓存机制** - 对VOD内容进行缓存，提升性能
5. **安全性增强** - URL验证、速率限制、CORS控制
6. **结构化日志** - 分级日志系统，便于监控
7. **健康检查** - 提供系统状态监控接口

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务
```bash
# 开发模式（需要安装nodemon）
npm run dev

# 生产模式
npm start
```

### 3. 使用代理
